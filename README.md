# Sales Performance Analysis (Power BI)

## Project Overview
This project analyzes sales data to evaluate business performance, revenue trends, and customer purchasing behavior using Power BI. The goal is to generate actionable insights that support decision-making in sales, operations, and product strategy.

## Problem Statement
Businesses need clear visibility into sales performance across products, locations, and time. Without structured analysis, it becomes difficult to identify key revenue drivers, customer behavior patterns, and the impact of discounts on profitability.

This project addresses:
- Which products generate the most revenue?
- When are sales highest?
- How do payment methods influence transactions?
- What is the relationship between discount and profit?

## Dataset Description
The dataset contains transactional sales records, including:
- Sales amount  
- Profit  
- Discount  
- Product category  
- Store location (country and city)  
- Payment method  
- Date (monthly and daily breakdown)  

## Methodology
- Cleaned and structured the dataset for consistency  
- Created calculated measures (Total Sales, Profit, Discount)  
- Performed time-based analysis (monthly and daily trends)  
- Analyzed category, location, and payment method performance  
- Built an interactive dashboard with slicers for dynamic filtering  

## Key Insights
- Total Sales: 4.14M  
- Total Profit: 3.54M  
- Home & Kitchen is the top-performing category  
- February recorded the highest monthly sales  
- Sales are stronger at the beginning of the week  
- Mobile payments contribute the highest transaction share
